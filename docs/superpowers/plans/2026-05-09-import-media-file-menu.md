# Import Media — File Menu Entry Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a "Import Media…" entry to the macOS File menu (after the standard New Item) that opens the existing media file picker, with ⌘O as its canonical shortcut.

**Architecture:** A new `Notification.Name.importMediaRequested` is posted by a `CommandGroup(after: .newItem)` button in `AppCommands`. `DocumentView` observes the notification and toggles its existing `showImporter` state, reusing the current `.fileImporter` plumbing. The ⌘O shortcut moves from the in-app button (which stays as a visual affordance without a shortcut) to the menu item, so the menu is the sole owner.

**Tech Stack:** SwiftUI `Commands` / `CommandGroup`, `NotificationCenter`, XCTest (unit + UI).

**Spec:** `docs/superpowers/specs/2026-05-09-import-media-file-menu-design.md`

Create `OnlyCueUITests/ImportMediaMenuTests.swift`:

```swift
import XCTest

final class ImportMediaMenuTests: XCTestCase {

    override func setUpWithError() throws {
        continueAfterFailure = false
    }

    /// Scenario: File menu surfaces media import
    /// Given the app is launched and a new document is open
    /// When the user clicks File → Import Media…
    /// Then the system open panel appears
    func test_fileMenu_importMedia_opensPicker() throws {
        let app = XCUIApplication()
        app.launch()

        // Reach a document window (DocumentGroup does not auto-open one).
        app.typeKey("n", modifierFlags: .command)

        let documentTitle = app.staticTexts["documentTitle"]
        XCTAssertTrue(
            documentTitle.waitForExistence(timeout: 5),
            "documentTitle should appear after ⌘N"
        )

        // Drive the menu bar: File → Import Media…
        let fileMenu = app.menuBarItems["File"]
        XCTAssertTrue(fileMenu.waitForExistence(timeout: 5), "File menu should exist")
        fileMenu.click()

        let importItem = app.menuItems["Import Media…"]
        XCTAssertTrue(
            importItem.waitForExistence(timeout: 2),
            "File menu should contain 'Import Media…'"
        )
        importItem.click()

        // The NSOpenPanel runs as a sheet; assert one of its standard buttons is present.
        let cancelButton = app.dialogs.buttons["Cancel"].firstMatch
        XCTAssertTrue(
            cancelButton.waitForExistence(timeout: 5),
            "system open panel should appear after activating Import Media…"
        )
        cancelButton.click()
    }
}
```
